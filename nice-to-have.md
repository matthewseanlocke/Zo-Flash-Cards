# Nice-to-Have Features

Future feature ideas for Zo Flash Cards. Pick one at a time to implement!

## New Content Types
- [x] **Shapes** - Circle, Square, Triangle, Rectangle, Star, Heart, Oval, Diamond, Pentagon, Hexagon *(in progress)*
- [ ] **Animals** - Common animals with pictures/emojis
- [ ] **Sight Words** - Common early reading words (the, and, is, it, etc.)
- [ ] **Fruits & Vegetables** - Apple, Banana, Carrot, etc.
- [ ] **Body Parts** - Eyes, Nose, Mouth, Hands, etc.
- [ ] **Emotions/Feelings** - Happy, Sad, Angry, Surprised (with emoji faces)
- [ ] **Opposites** - Big/Small, Hot/Cold, Up/Down pairs
- [ ] **Days of the Week / Months**
- [ ] **Weather** - Sunny, Rainy, Cloudy, Snowy

## Learning Enhancements
- [ ] **Audio Pronunciation** - Tap to hear letter/word spoken
- [ ] **Letter Sounds (Phonics)** - "A says 'ah'" not just letter names
- [ ] **Counting Objects** - Show 3 apples, child identifies "3"
- [ ] **Simple Math** - Basic addition/subtraction with visuals
- [ ] **Letter Tracing** - Guided path instead of free coloring
- [ ] **Uppercase/Lowercase Matching** - Match A to a

## Gamification
- [ ] **Rewards/Stickers** - Earn stickers for good sessions
- [ ] **Streaks** - Track daily practice streaks
- [ ] **Achievements/Badges** - "Learned all letters!" etc.
- [ ] **Time Challenge Mode** - Beat the clock
- [ ] **Difficulty Levels** - Easy/Medium/Hard

## App Features
- [ ] **Multiple Profiles** - Support multiple children
- [ ] **Progress Charts** - Visual progress over time
- [ ] **Custom Card Sets** - Parents add their own cards
- [ ] **Sound Effects** - Fun sounds on correct/wrong
- [ ] **Themes** - Different background colors/styles
